 Sistema de Cadastro de Livros — Biblioteca Escolar

Sistema web desenvolvido em PHP + MySQL como desafio prático do curso de Desenvolvimento de Sistemas.

O objetivo do projeto é substituir o controle manual em papel da biblioteca escolar por um sistema digital simples, permitindo cadastro e consulta de livros.

 Objetivo do Projeto

Desenvolver um mini-sistema capaz de:

 Cadastrar livros no banco de dados (INSERT)

 Listar todos os livros cadastrados (SELECT)

 Filtrar livros por título ou categoria (SELECT com WHERE)

 Organizar telas separadas (cadastro e listagem)

 Aplicar regras básicas de negócio

 Tecnologias Utilizadas

PHP

MySQL

HTML5

CSS3

XAMPP / WAMP (ambiente local)

phpMyAdmin
